# Example Input: roadmap-writer

## Scenario A: Roadmap mode (multiple epics)

Build a Q2-Q3 roadmap for the workspace product.

Context:
- We need 3 epics max.
- Keep plans initiative-first, not date-locked.
- Include dependencies and links to existing PRDs where available.
- Prioritization context: we mostly need scope clarity, not detailed effort modeling yet.
- Each epic should make the primary JTBD explicit.

Epic candidates:
- Improve onboarding activation: when a first-time admin creates a workspace, they want to finish setup quickly and safely, so they can start collaborating right away.
- Strengthen reporting reliability: when analysts review weekly performance, they want trustworthy dashboards, so they can make decisions without manual reconciliation.
- Launch role-based admin controls: when workspace owners delegate admin tasks, they want clear role boundaries, so they can share work without losing governance.
