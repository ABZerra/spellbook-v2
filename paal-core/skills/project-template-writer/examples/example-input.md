# Example Input: project-template-writer

Create a project template for cross-functional product launches that is tool-agnostic and can be instantiated in Notion, Word, Excel, and Jira.

Constraints:
- Canonical entities must include Vision, Roadmap, Initiatives, and Resources.
- The template must preserve the original Project Kickstart baseline from report and screenshot.
- The template must support CSV import and export.
- Include exact default Notion views: Roadmap (`Milestones`, `Timeline`, `Table`), Initiatives (`Kanban`, `Table`), Resources (`Resource Table`, `Sorted A-Z List`).
- Include exact page block order and seed sample records matching the baseline.
- Keep real-time sync and automation out of v1.
